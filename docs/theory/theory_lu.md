# LU Decomposition — Theory

LU factorization decomposes:

\[
PA = LU
\]

where:

- \( P \) is a permutation matrix  
- \( L \) is lower triangular  
- \( U \) is upper triangular  

---

# 1. Solving a System

1. Solve \( Ly = Pb \)  
2. Solve \( Ux = y \)

---

# 2. Requirements

- Pivoting required for stability  
- Matrix must be non‑singular
