# Cholesky Factorization — Theory

Cholesky decomposes a symmetric positive‑definite matrix:

$$
A = LL^T
$$

---

# 1. Solving a System

1. Solve $Ly = b$  
2. Solve $L^T x = y$

---

# 2. Requirements

- Matrix must be symmetric  
- Matrix must be positive definite  

---

# 3. Conceptual Example

$$
x = (1, 2)
$$
