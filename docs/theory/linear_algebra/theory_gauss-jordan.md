# Gauss‑Jordan Elimination — Theory

Gauss‑Jordan solves:

$$
Ax = b
$$

by reducing the augmented matrix $[A|b]$ to **reduced row echelon form**.

---

# 1. Steps

1. Pivot selection  
2. Normalize pivot row  
3. Eliminate all other rows  

---

# 2. Properties

- Produces exact solution  
- Also computes inverse when applied to $[A|I]$

---

# 3. Conceptual Example

Same system:

$$
x = (1, 2)
$$
