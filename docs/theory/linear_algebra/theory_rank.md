# Matrix Rank — Theory

The rank of a matrix is the number of linearly independent rows or columns.

---

# 1. Interpretation

- Rank = dimension of column space  
- Rank = number of pivots in row‑echelon form  
- Rank < n ⇒ matrix is singular  

---

# 2. Conceptual Example

$$
A = \begin{pmatrix}
4 & 3 \\
6 & 3
\end{pmatrix}
$$

Rank:

$$
\operatorname{rank}(A) = 2
$$
